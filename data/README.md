# Data Directory

This directory is intended for storing data files for the Week 1 Workshop.

You can place various types of data files here such as:
- JSON files
- CSV files
- Text files
- Images
- Any other data assets needed for the workshop

Example structure:
```
data/
├── README.md (this file)
├── sample.json
├── data.csv
└── images/
    └── example.jpg
```
